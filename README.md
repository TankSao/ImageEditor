# ImageEditor
仿微信，长按图片弹出选择框以及编辑图片保存</br>
之前上传一个长按识别二维码的，可以判断图片是否有二维码，从而是否显示识别图中二维码以及识别功能</br>
该demo实现的功能有保存图片以及编辑（简单的涂鸦，可选择多种颜色）</br>
